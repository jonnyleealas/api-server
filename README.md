# Step 1:
Open terminal and from your directory run the following commands.
```
1. npm i
2. touch.env
```
# Step 2:
Insert the following inside of your .env file
```
PORT=<openport>
DATABASE_URL=db-url

```
# Step 3:
The following restful commands can be implemented to manipulate data.
```
GET
POST
PUT
DELETE


```
